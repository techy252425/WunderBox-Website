Native macOS App

WunderBox

Eine native Menüleisten-App für macOS mit privilegiertem Helper (SMJobBless), Sensor-Events und flexiblen Audio-Modi — gebaut für Apple-Silicon-MacBooks.

GitHub Repo öffnen Features

Highlights

Menüleisten-App

Leichtgewichtige SwiftUI-UI für Status, Steuerung und Event-History.

Privilegierter Helper

XPC-Helper via SMJobBless für Sensorzugriff mit sauberem App/Helper-Split.

Audio-Modi

Pain, Sexy, Halo, Lizard, Custom, Imported, Mix und Lid-Creak.

Simulation Mode

Läuft auch auf Geräten ohne unterstützten Sensor.

Technik

  • SwiftUI App + XPC Helper + Shared Contracts
  • SMJobBless für echte Helper-Installation
  • XcodeGen-Projektstruktur
  • Audio-Assets direkt im Repo gebündelt

Quick Start

  • cd macos
  • xcodegen generate
  • xcodebuild -project WunderBoxMac.xcodeproj -scheme WunderBoxApp -configuration Debug CODE_SIGNING_ALLOWED=NO build
