# WunderBox Website

Premium Landing Page für die [WunderBox](https://github.com/Anes-03/WunderBox) macOS App.

## Dateien

- `index.html` — Hauptseite (Hero, Features, Architektur, Sicherheit, FAQ, CTA)
- `styles.css` — Styling (responsive, glassmorphism, dark theme)
- `script.js` — Scroll-Reveal-Animationen, FAQ-Accordion, Mobile-Nav

## Lokal testen

```bash
python3 -m http.server 8080
# dann öffnen: http://localhost:8080
```

## Technologie

Rein statische Website — kein Framework, kein Build-Step. HTML + CSS + minimales JS.
